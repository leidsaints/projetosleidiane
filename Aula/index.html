É interessante saber como a web funciona e navegar pela Internet não é nada mais nada menos do que comunicar seu dispositivo seja ele um computador, um notebook, uma smarTV, um celular, etc, á outro dispositivo chamado servidor, por meio da rede de dados que corresponde à internet. Então basicamente há dois lados nessa história, o do usuário da internet, que representa o client-side ou lado do cliente, e o servidor, que é o server-side cliente e servidor se comunicam por meio de requisições e respostas, o cliente manda requisições para o servidor e o servidor responde com o que o cliente requisitou.

Oque são servers

voltar

Um servidor, em suma, é um computador com grande capacidade de armazenamento e processamento cujo objetivo é conectar outros computadores através de uma rede. Os servidores processam e executam solicitações feitas por usuário através de softwares, bancos de dados, envio e recebimento de informações como e-mails, envio de formulários, hospedagem de websites e outros.Um servidor funciona, grosso modo, como um grande computador, mas ao invés de executar operações em nível doméstico, ele o faz em grande escala, transferindo esses resultados para diferentes computadores.

" Existem, hoje em dia, diversos tipos de servidores no mercado. Dos mais populares, como os de e-mail, aos mais complexos, como os de proxy, estes recursos estão presentes por toda a internet, servidores de e-mail são, provavelmente, os servidores mais comuns. São eles os responsáveis pelos serviços de hospedagem disponíveis hoje no mercado e neles são armazenados dezenas de milhões de informações em forma de e-mails, estes servidores nenhum e-mail pode ser enviado ou recebido na sua caixa de entrada, uma vez que é através deles que a transferência de informações entre um remetente e um destinatário é feita. Serviço de armazenagem em nuvem tornou-se muito popular nos últimos anos, mas, para que ele seja possível, é necessário um servidor na nuvem, com ele diferentes tarefas podem ser executadas e armazenadas remotamente, sem a necessidade de que haja um servidor dentro da empresa que o utilizará. Os servidores web, assim como os servidores de e-mail, são bastante comuns e servem para processar e executar aplicações que disponibilizam conteúdo na internet. Os servidores de banco de dados são aqueles utilizados para melhorar o desempenho do processamento e transferência de dados de redes que recebem um grande número de requisições. Serviços de armazenagem, por exemplo, são grandes utilizadores de servidores de bancos de dados, porque precisam arquivar com segurança grande número de dados e informações. Os servidores proxy aumentam a eficiência e a segurança dos acessos à internet por usuários comuns. Eles funcionam como filtros que intermediam as requisições dos clientes e o recebimento dessas informações. Em outras palavras, quando um usuário faz uma requisição na web, ela é recebida por um servidor proxy e, então, é este servidor se comunica com a internet retornando a informação esperada. Os servidores de aplicativo são os ambientes que proporcionam a execução eficiente dos aplicativos que utilizamos. São nestes ambientes que são armazenadas as informações e executadas as requisições dos usuários, tornando possível que estas aplicações desempenhem suas funções com efetividade. Os servidores FTP são servidores muito importantes porque são eles que permitem que arquivos sejam transferidos, através de download e upload, principalmente em ambientes cloud. Eles funcionam via conexão com o protocolo de transferência de arquivo FTP (File Transfer Protocol) e proporcionam um ambiente seguro para a troca de arquivos. "

Estrutura básica de um HTML?

voltar

A estrutura do HTML consiste em uma série de elementos ou tags, que são marcadores especiais inseridos no texto para definir o significado e a função de diferentes partes do conteúdo. Essas tags são interpretadas pelos navegadores da web para renderizar a página conforme a estrutura e o estilo especificados, As tags HTML são utilizadas para delimitar elementos específicos na página. Cada tag é composta por um nome e pode conter atributos e/ou conteúdo entre as tags de abertura e fechamento. As tags são escritas entre colchetes angulares, por exemplo: .Elementos:Um elemento HTML é formado pela combinação de uma tag de abertura, o conteúdo do elemento e uma tag de fechamento.

Listas ordenadas e não ordenadas

voltar

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados a lista ordenada enumera os elementos, a lista não-ordenada coloca marcadores antes de cada elemento. Em uma lista ordenada é delimitada pelas etiquetas OL (Ordered List). Cada item da lista é delimitado pelas etiquetas LI (List Item). A estrutura da lista não-ordenada é semelhante a da lista ordenada porém, ao invés de ser construída com o elemento OL, é construída com o elemento UL (Unordered List).
